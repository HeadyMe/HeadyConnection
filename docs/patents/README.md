# Patents & Invention Disclosures

This directory tracks invention disclosures and provisional applications.

- Invention disclosures: `docs/patents/invention_disclosures/`
- Provisional applications: `docs/patents/provisional/`

See `docs/patents/INVENTION_DISCLOSURE_INDEX.md` for the current register.

## Filing Queue (Draft)
See `docs/patents/FILING_QUEUE.md` for the current draft filing order and review status.

## Provisional Templates
When a new invention is pending a finalized system name, use the Patent Application 40 template as a placeholder and replace the bracketed fields before counsel review.

## Draft Scaffolding Helper
Use `scripts/patents/scaffold_patent.py` to generate placeholder IDF + DOCX pairs for future filings while keeping index consistency.
