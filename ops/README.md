# Operations

This directory contains operational scaffolding for development and production deployments.

- `ops/dev`: local workstation tooling (DDEV).
- `ops/prod`: bare metal deployment notes for Nginx + PHP-FPM behind Cloudflare Tunnel.
- `ops/cloudflare`: tunnel configuration templates and guidance.
- `ops/compose`: docker-compose scaffolding for ops tooling (e.g., Uptime Kuma).
- `ops/systemd`: systemd unit scaffolding for long-running services.
